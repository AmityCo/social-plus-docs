<Note>
  **Xcode 26 compatibility notice**

  Beginning with **Xcode 26**, changes in the Swift compiler break forward compatibility for\
  **RealmSwift** binaries built with earlier Xcode versions, even though library evolution is enabled. Historically, SDK integrations have been forward compatible across Xcode versions.\
  We are actively working toward restoring this experience and have started planning\
  a migration away from Realm. In the meantime, we recommend aligning your projectâ€™s **Xcode version** with the Xcode version  used to build the **SDK**. We appreciate your patience as we improve long-term compatibility.
</Note>
