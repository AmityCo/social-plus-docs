## Channel unread count

The SDK provides a simple way for clients to retrieve the unread count for a channel. To view the unread count for a channel, we can get it from a channel object. This count represents the number of messages that you have not yet read in that channel.

<Tabs>
  <Tab title="iOS">
    ```swift
    func getChannelHasUnreadMention(channel: AmityChannel) {
        channel.isMentioned
    }
    ```
  </Tab>
  <Tab title="Android">
    ```kotlin
    fun getChannelUnreadCount(channel: AmityChannel) {
    		channel.getUnreadCount()
    }
    ```
  </Tab>
  <Tab title="TypeScript">
    ```typescript
    // handleLiveObject will be triggered every time the live channel is updated.
    const handleLiveObject = (liveObject: Amity.LiveObject<Amity.Channel>) => {
      const channel = liveObject.data;
    
      // An example of a reading unread count is in the line below.
      console.log(`channel ID ${channel.channelId} has a current unread count: ${channel.unreadCount}`);
    };
    ```
  </Tab>
</Tabs>

## Total unread count of all channels

Gets the total unread count of all channels. This only includes unread count from default subChannel of the channels that are already cached after being fetched from the server and excludes all.

<Tabs>
  <Tab title="iOS">
    ```swift
    var disposeBag: Set<AnyCancellable> = []
    
    func observeTotalChannelsUnreadCount() {
        channelRepository.getTotalChannelsUnread().sink(receiveValue: { unread in
            print("Channels total unread count: \(unread.unreadCount)")
            print("User's mentioned status: \(unread.isMentioned)")
        }).store(in: &disposeBag)
    }
    ```
  </Tab>
  <Tab title="Android">
    ```kotlin
    fun getChannelUnreadCount(channel: AmityChannel) {
    		channel.getUnreadCount()
    }
    ```
  </Tab>
  <Tab title="TypeScript">

  </Tab>
</Tabs>
