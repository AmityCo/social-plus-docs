## Feature Overview

The Unread Status system tracks message read and delivery states in real time, with accurate unread indicators, delivery confirmations, and user activity tracking.

## Feature Integration Patterns

<AccordionGroup>
  <Accordion title="Chat List Integration" icon="list">
    **Display unread status in channel lists**
    
    Create compelling channel list interfaces that clearly communicate unread activity:
    - **Unread Badges**: Show exact counts with visual hierarchy for mentions
    - **Visual Indicators**: Use colors, typography, and spacing to indicate priority
    - **Last Message Preview**: Combine unread counts with message previews
    - **Sorting Logic**: Prioritize channels with unread messages and mentions
    
    Effective chat list integration helps users quickly identify where their attention is needed.
  </Accordion>

  <Accordion title="Message Interface Integration" icon="message">
    **Embed status tracking in message displays**
    
    Integrate read and delivery status directly into message interfaces:
    - **Read Receipts**: Show checkmarks or indicators for read status
    - **Delivery Confirmations**: Display delivery status for sent messages
    - **User Lists**: Show which users have read or received messages
    - **Real-time Updates**: Update status indicators as they change
    
    Message-level integration provides immediate feedback on communication effectiveness.
  </Accordion>

  <Accordion title="Analytics Integration" icon="chart-simple">
    **Leverage status data for insights**
    
    Use unread and delivery data for user engagement analytics:
    - **Response Metrics**: Track average response times and engagement rates
    - **User Activity**: Monitor reading patterns and message interaction
    - **Channel Performance**: Identify most and least active channels
    - **Mention Analytics**: Track mention frequency and response rates
    
    Analytics integration provides valuable insights into communication patterns and user behavior.
  </Accordion>
</AccordionGroup>

## Getting Started

<Steps>
  <Step title="Implement Channel Unread Count">
    Start with [Channel Unread Count](/social-plus-sdk/chat/engagement-features/unread-status/channel-unread-count) to display unread indicators in your chat list and navigation.
  </Step>
  
  <Step title="Set Up Receipt Sync">
    Configure [Message Receipt Sync](/social-plus-sdk/chat/engagement-features/unread-status/message-receipt-sync) for real-time read and delivery tracking in active channels.
  </Step>
  
  <Step title="Add Read Status Tracking">
    Implement [Message Read Status](/social-plus-sdk/chat/engagement-features/unread-status/message-read-status) to mark messages as read and maintain accurate unread counts.
  </Step>
  
  <Step title="Enable Delivery Tracking">
    Add [Message Delivery Status](/social-plus-sdk/chat/engagement-features/unread-status/message-delivery-status) for comprehensive delivery confirmation and user analytics.
  </Step>
</Steps>

<Tip>
**Implementation Strategy**: Begin with basic unread count display, then progressively add receipt sync, read tracking, and delivery status. Focus on user experience and performance optimization as you build out comprehensive status tracking capabilities.
</Tip>
