---
title: "Get Latest Comment"
description: "Retrieve the most recent comment for any reference with real-time updates and live object support"
---

Social Plus SDK provides the `getLatestComment` method to fetch the most recent comment for any reference within your application. This feature enhances user experience by keeping users informed about the latest interactions and fostering a more dynamic, interactive environment with real-time updates.

<CardGroup cols={2}>
  <Card title="Live Object Integration" icon="bolt">
    Returns live objects that automatically update when new comments arrive
  </Card>
  <Card title="Optimized Queries" icon="gauge-high">
    Efficient single-comment retrieval with minimal network overhead
  </Card>
</CardGroup>

<Info>
The `getLatestComment` method returns a Live Object that you can observe for real-time updates when Realtime Events are subscribed, ensuring your app stays current with the latest community interactions.
</Info>

### Parameters

| Parameter | Type | Required | Description |
|-----------|------|----------|-------------|
| `referenceId` | String | Yes | ID of the reference (post, story, etc.) |
| `referenceType` | Enum | Yes | Type of reference (e.g., `.post`, `.story`) |
| `includeReplies` | Boolean | No | Whether to include reply comments (default: false) |

<CodeGroup>
```swift iOS
var token: AmityNotificationToken?

func getLatestCommentExample() async {
    do {
        let comment = try await commentRepository.getLatestComment(
            withReferenceId: "<post-id>",
            referenceType: .post,
            includeReplies: true
        )
    } catch {
        // Handle error here
    }
}
```

```kotlin Android
fun getLatestComment(
    commentRepository: AmityCommentRepository,
    postId: String
) {
    commentRepository.getLatestComment()
        .post(postId = postId)
        .build()
        .query()
        .doOnSuccess { comment: AmityComment ->
            // AmityComment
        }
        .doOnError {
            // AmityException
        }
        .subscribe()
}
```
