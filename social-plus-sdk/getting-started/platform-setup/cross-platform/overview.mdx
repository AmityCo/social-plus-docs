# Cross-platform Development

Build apps that run on both iOS and Android with a single codebase. Social Plus provides comprehensive SDKs for React Native and Flutter, enabling you to deliver native performance with cross-platform efficiency.

## Cross-platform Advantages

### Code Sharing Benefits
- **Single Codebase**: Write once, deploy to both iOS and Android
- **Shared Business Logic**: Consistent behavior across platforms
- **Unified Development**: Same team can build for both platforms
- **Faster Time-to-Market**: Reduce development and testing time

### Native Performance
- **Native Compilation**: Flutter compiles to native ARM code
- **Native Bridge**: React Native uses native modules for performance
- **Native UI**: Platform-specific UI components and interactions
- **Native APIs**: Access to device features and platform services

```

│           JavaScript Layer          │

│  │     Your App Code (JS/TS)      │ │
│  │  ┌─────────────────────────────┐ │ │
│  │  │    Social Plus SDK (JS)    │ │ │
│  │  └─────────────────────────────┘ │ │
│  └─────────────────────────────────┘ │
├─────────────────────────────────────┤
│           Bridge Layer              │
├─────────────────────────────────────┤
│          Native Layer               │
│  ┌─────────────────┬───────────────┐ │
│  │   iOS Native    │ Android Native│ │
│  │   (Objective-C/ │  (Java/Kotlin)│ │
│  │    Swift)       │               │ │
│  └─────────────────┴───────────────┘ │
└─────────────────────────────────────┘
```

## Development Best Practices

### Code Organization
- **Feature-based Structure**: Organize code by features, not by file types
- **Shared Components**: Create reusable UI components
- **Platform-specific Code**: Use platform checks when needed
- **State Management**: Implement consistent state management patterns

### Performance Optimization
- **List Virtualization**: Use FlatList (RN) or ListView.builder (Flutter)
- **Image Optimization**: Implement proper image caching and resizing
- **Memory Management**: Dispose of resources and subscriptions properly
- **Network Efficiency**: Batch API calls and implement proper caching

### Testing Strategy
- **Unit Tests**: Test business logic and utility functions
- **Widget Tests**: Test UI components in isolation (Flutter)
- **Component Tests**: Test React components (React Native)
- **Integration Tests**: Test complete user workflows
- **Platform Testing**: Test on both iOS and Android devices

## React Native Overview

The Social+ TypeScript SDK is fully compatible with React Native, enabling you to build cross-platform apps with native performance and a unified codebase.

<Info>
For older JavaScript runtimes in React Native, add a polyfill for `Object.fromEntries`:

1. Install core-js:
```bash
npm install core-js
```
2. Import the polyfill at the root of your project:
```javascript
import 'core-js/features/object/from-entries';
```
</Info>
