Social+ provides powerful SDKs for adding social features like chat, feeds, communities, and live video to your applications. This guide gets you up and running quickly.


<Steps>
<Step title="Choose Your Platform">
Select the SDK that matches your development environment:

<CardGroup cols={2}>
<Card title="Mobile Apps" icon="mobile">
- [iOS SDK](/social-plus-sdk/getting-started/install-ios)
- [Android SDK](/social-plus-sdk/getting-started/install-android)
- [React Native SDK](/social-plus-sdk/getting-started/install-react-native)
- [Flutter SDK](/social-plus-sdk/getting-started/install-flutter)
</Card>

<Card title="Web Apps" icon="globe">
- [TypeScript SDK](/social-plus-sdk/getting-started/install-web)
- [JavaScript SDK (Deprecated)](/social-plus-sdk/getting-started/install-web#javascript-sdk-deprecated)
</Card>
</CardGroup>
</Step>

<Step title="Get Your API Key">
1. Visit the [Social+ Console](https://portal.social.plus/login)
2. Navigate to **Settings** â†’ **Security**
3. Copy your API key from the **Keys** section

<Note>
Keep your API key secure. Never expose it in client-side code in production.
</Note>
</Step>

<Step title="Initialize the SDK">
After installation, initialize the SDK with your API key:

<CodeGroup>
```swift iOS
let client = try! AmityClient(apiKey: "your-api-key", region: .SG)
```

```kotlin Android
AmityClient.initialize(
    apiKey = "your-api-key",
    region = AmityRegion.SG
)
```

```typescript TypeScript
import { Client } from '@amityco/ts-sdk';

const client = Client.create({
  apiKey: 'your-api-key',
  region: 'sg'
});
```

```dart Flutter
AmityClient.initialize(
  apiKey: 'your-api-key',
  region: AmityRegion.SG
);
```
</CodeGroup>
</Step>

<Step title="Authenticate Users">
Log in users to access Social+ features:

<CodeGroup>
```swift iOS
client.login(
    userId: "user123",
    displayName: "John Doe",
    authToken: "your-auth-token"
) { success, error in
    if success {
        print("Login successful")
    }
}
```

```kotlin Android
AmityClient.login(
    userId = "user123",
    displayName = "John Doe",
    authToken = "your-auth-token"
)
```

```typescript TypeScript
await client.login({
  userId: 'user123',
  displayName: 'John Doe',
  authToken: 'your-auth-token'
});
```

```dart Flutter
await AmityClient.login(
    userId: 'user123',
    displayName: 'John Doe',
    authToken: 'your-auth-token'
);
```
</CodeGroup>
</Step>
</Steps>
