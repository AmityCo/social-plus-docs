# Installation Guide

Choose your platform and installation method to get started with Social+ UIKit. We support both **package installation** for quick setup and **GitHub forking** for complete customization.

<Info>
**Two Installation Approaches**: Use package installation for quick integration, or fork from GitHub for complete control and customization.
</Info>

## Package Installation

Install UIKit through package managers for quick integration:

<Tabs>
  <Tab title="iOS">
    ### Prerequisites
    - iOS 12.0 or later
    - Xcode 12.0 or later
    - Swift 5.0 or later

    ### Installation Steps

    <Tabs>
      <Tab title="SwiftPM (Recommended)">
        **Add Package Dependency in Xcode:**

        1. Open your project in Xcode
        2. Go to **File â†’ Add Package Dependencies**
        3. Enter the repository URL:
        ```
        https://github.com/AmityCo/Amity-Social-Cloud-UIKit-iOS-SwiftPM
        ```
        4. Select **Exact Version** for beta releases
        5. Choose the latest version and add to your target

        **Import in your code:**
        ```swift
        import AmityUIKit
        ```
      </Tab>
      <Tab title="CocoaPods">
        **Add to your Podfile:**
        ```ruby
        pod 'AmityUIKit', '~> 4.0'
        ```

        **Install dependencies:**
        ```bash
        pod install
        ```

        **Import in your code:**
        ```swift
        import AmityUIKit
        ```
      </Tab>
    </Tabs>

    ### Initialize UIKit
    ```swift
    // In your AppDelegate or SceneDelegate
    AmityUIKitManager.setup(
        apiKey: "YOUR_API_KEY",
        region: .US // or .EU, .SG
    )
    ```
  </Tab>

  <Tab title="Android">
    ### Prerequisites
    - Android 5.0 (API Level 21) or higher
    - Kotlin support enabled
    - Data binding enabled

    ### Gradle Configuration

    **Project-level build.gradle:**
    ```gradle
    allprojects {
        repositories {
            google()
            mavenCentral()
            maven { url 'https://jitpack.io' }
        }
    }
    ```

    **App-level build.gradle:**
    ```gradle
    android {
        compileSdk 34
        
        defaultConfig {
            minSdk 21
            targetSdk 34
        }
        
        buildFeatures {
            dataBinding true
        }
        
        compileOptions {
            sourceCompatibility JavaVersion.VERSION_1_8
            targetCompatibility JavaVersion.VERSION_1_8
        }
    }

    dependencies {
        implementation 'co.amity.socialplusuikit:uikit:4.0.0'
    }
    ```

    ### Initialize UIKit
    ```kotlin
    // In your Application class
    class MyApplication : Application() {
        override fun onCreate() {
            super.onCreate()
            
            AmityUIKitClient.setup(
                context = this,
                apiKey = "YOUR_API_KEY",
                region = AmityRegion.US // or EU, SG
            )
        }
    }
    ```
  </Tab>

  <Tab title="Web (React)">
    ### Prerequisites
    - React 16.8 or later
    - Node.js 14 or later
    - Modern browser support

    ### Installation
    ```bash
    npm install @amityco/ui-kit-react
    # or
    yarn add @amityco/ui-kit-react
    ```

    ### Setup Provider
    ```jsx
    import { AmityUIKitProvider } from '@amityco/ui-kit-react';

    function App() {
      return (
        <AmityUIKitProvider
          apiKey="YOUR_API_KEY"
          apiRegion="us" // or "eu", "sg"
          userId="user123"
        >
          {/* Your app components */}
        </AmityUIKitProvider>
      );
    }
    ```

    ### Import Styles
    ```jsx
    import '@amityco/ui-kit-react/dist/index.css';
    ```
  </Tab>

  <Tab title="Flutter">
    ### Prerequisites
    - Flutter 3.0 or later
    - Dart 2.17 or later
    - iOS 11.0+ / Android API 21+

    ### Installation
    ```yaml
    dependencies:
      amity_uikit_beta_service: ^0.2.0
    ```

    ### Platform Configuration

    **iOS (ios/Podfile):**
    ```ruby
    platform :ios, '11.0'
    ```

    **Android (android/app/build.gradle):**
    ```gradle
    android {
        compileSdkVersion 34
        minSdkVersion 21
        targetSdkVersion 34
    }
    ```

    ### Initialize UIKit
    ```dart
    import 'package:amity_uikit_beta_service/amity_uikit_beta_service.dart';

    await AmityUIKit.setup(
      apikey: "YOUR_API_KEY",
      region: AmityRegion.US, // or EU, SG
    );
    ```
  </Tab>
</Tabs>

## GitHub Fork Installation

For complete customization and control over the UIKit source code:

<Warning>
**React Native Note**: React Native UIKit is only available through GitHub forking, not package installation.
</Warning>

<Tabs>
  <Tab title="iOS">
    ### Fork Repository
    1. Fork the [iOS UIKit repository](https://github.com/AmityCo/Amity-Social-Cloud-UIKit-iOS-OpenSource)
    2. Clone your forked repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-iOS-OpenSource.git
    ```

    ### Integration Options

    <Tabs>
      <Tab title="Git Submodule">
        **Add as submodule:**
        ```bash
        cd YourProject
        git submodule add https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-iOS-OpenSource.git AmityUIKit
        git submodule update --init --recursive
        ```

        **Add to Xcode:**
        1. Drag `AmityUIKit.xcodeproj` into your project
        2. Add `AmityUIKit` framework to your target dependencies
        3. Import: `import AmityUIKit`
      </Tab>
      <Tab title="Direct Integration">
        **Copy source files:**
        1. Copy the `UpstraUIKit` folder to your project
        2. Add files to your Xcode project
        3. Configure build settings and dependencies
        4. Import: `import UpstraUIKit`
      </Tab>
    </Tabs>

    ### Customization Benefits
    - Modify UI components directly
    - Add custom business logic
    - Implement custom themes
    - Full control over dependencies
  </Tab>

  <Tab title="Android">
    ### Fork Repository
    1. Fork the [Android UIKit repository](https://github.com/AmityCo/Amity-Social-Cloud-UIKit-Android-OpenSource)
    2. Clone your forked repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-Android-OpenSource.git
    ```

    ### Integration Methods

    <Tabs>
      <Tab title="Git Submodule">
        **Add as submodule:**
        ```bash
        cd YourAndroidProject
        git submodule add https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-Android-OpenSource.git social-ui-kit
        ```

        **Include in settings.gradle:**
        ```gradle
        include ':social-ui-kit'
        project(':social-ui-kit').projectDir = new File('social-ui-kit')
        ```

        **Add dependency:**
        ```gradle
        dependencies {
            implementation project(':social-ui-kit')
        }
        ```
      </Tab>
      <Tab title="Local Module">
        **Copy and import:**
        1. Copy the UIKit module to your project
        2. Include in `settings.gradle`
        3. Add as dependency in `build.gradle`
        4. Sync project
      </Tab>
    </Tabs>

    ### Customization Benefits
    - Modify layouts and styling
    - Add custom fragments/activities
    - Implement custom data binding
    - Control build configurations
  </Tab>

  <Tab title="React Native">
    <Info>
    **React Native UIKit is only available through GitHub forking** - no package installation option.
    </Info>

    ### Fork Repository
    1. Fork the [React Native UIKit repository](https://github.com/AmityCo/Amity-Social-Cloud-UIKit-React-Native-OpenSource)
    2. Clone your forked repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-React-Native-OpenSource.git
    ```

    ### Integration Steps
    ```bash
    cd YourReactNativeProject
    
    # Add as git submodule
    git submodule add https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-React-Native-OpenSource.git amity-react-native-social-ui-kit
    
    # Install dependencies
    cd amity-react-native-social-ui-kit
    npm install
    
    # Link in your main project
    cd ..
    npm install ./amity-react-native-social-ui-kit
    ```

    ### Platform Setup

    **iOS Dependencies (ios/Podfile):**
    ```ruby
    platform :ios, '11.0'

    pod 'AmitySDK', '~> 6.0.0'
    pod 'Realm', '10.20.2'
    ```

    **Android Dependencies:**
    ```gradle
    dependencies {
        implementation 'co.amity:amity-sdk:6.0.0'
        implementation 'io.realm:realm-android-library:10.20.2'
    }
    ```

    ### Usage
    ```jsx
    import AmitySocialUIKit from './amity-react-native-social-ui-kit';

    export default function App() {
      return (
        <AmitySocialUIKit
          apiKey="YOUR_API_KEY"
          apiRegion="us" // or "eu", "sg"
          userId="user123"
        />
      );
    }
    ```

    ### Customization Benefits
    - Full React Native component access
    - Custom styling and theming
    - Modified navigation flows
    - Custom business logic integration
  </Tab>

  <Tab title="Web (React)">
    ### Fork Repository

    2. Clone your forked repository:

        2. Install required dependencies
        3. Configure build tools

      </Tab>
    </Tabs>

    ### Customization Benefits
    - Modify React components
    - Custom CSS and styling
    - Add custom hooks and utilities
    - Control routing and state management
  </Tab>

  <Tab title="Flutter">
    ### Fork Repository
    1. Fork the [Flutter UIKit repository](https://github.com/AmityCo/Amity-Social-Cloud-UIKit-Flutter-OpenSource)
    2. Clone your forked repository:
    ```bash
    git clone https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-Flutter-OpenSource.git
    ```

    ### Integration Steps
    **Add to pubspec.yaml:**
    ```yaml
    dependencies:
      amity_uikit:
        path: ./Amity-Social-Cloud-UIKit-Flutter-OpenSource
    ```

    **Or as git dependency:**
    ```yaml
    dependencies:
      amity_uikit:
        git:
          url: https://github.com/YOUR_USERNAME/Amity-Social-Cloud-UIKit-Flutter-OpenSource.git
          ref: main
    ```

    ### Customization Benefits
    - Modify Flutter widgets
    - Custom themes and styling
    - Add custom business logic
    - Control state management
  </Tab>
</Tabs>
