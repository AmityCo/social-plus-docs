## üÜò Having Issues? Quick Fixes

<AccordionGroup>
  <Accordion title="‚ùå 'API Key Invalid' Error" icon="key">
    **Problem:** Getting authentication errors after setup
    
    **Solution:**
    1. Check your API key at [console.socialplus.com](https://console.socialplus.com)
    2. Ensure you're using the correct region (`us`, `eu`, or `sg`)
    3. Verify your bundle ID/package name matches your console settings
    
    ```javascript
    // Make sure region matches your console setup
    AmityUIKit.setup({
      apiKey: 'YOUR_API_KEY',
      region: 'us' // Change this to match your region
    });
    ```
  </Accordion>

  <Accordion title="üì± Components Not Showing" icon="eye-slash">
    **Problem:** UIKit components appear blank or don't render
    
    **Solution:**
    1. Ensure you've called `AmityUIKit.setup()` before rendering components
    2. Check that you have an active internet connection
    3. Verify your user is authenticated
    
    ```javascript
    // Initialize BEFORE using components
    AmityUIKit.setup({ apiKey: 'YOUR_API_KEY' });
    
    // Then render components
    <AmityFeedComponent />
    ```
  </Accordion>

  <Accordion title="üîß Build Errors" icon="wrench">
    **Problem:** Getting compilation or build errors
    
    **Platform-specific solutions:**
    - **iOS:** Clean build folder (Cmd+Shift+K), then rebuild
    - **Android:** Run `./gradlew clean` then rebuild
    - **React Native:** Clear metro cache: `npx react-native start --reset-cache`
    - **Flutter:** Run `flutter clean && flutter pub get`
  </Accordion>
</AccordionGroup>
