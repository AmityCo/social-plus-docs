# Stories Components

Social+ UIKit provides powerful story components that enable users to create, share, and view ephemeral content. The story feature enhances engagement and fosters community interaction by allowing users to capture and share moments within their communities.

## Key Features

### Story Creation
- **Media Capture**: Built-in camera functionality for photos and videos
- **Gallery Selection**: Pick existing media from device gallery
- **Story Drafting**: Edit and enhance content before publishing
- **Target Selection**: Choose specific communities or audiences
- **Privacy Controls**: Manage who can view your stories

### Story Viewing
- **Immersive Interface**: Full-screen story viewing experience
- **Auto-Progression**: Automatic advancement through story sequences
- **Interactive Elements**: Tap to pause, swipe to navigate
- **Engagement Tools**: Comments, reactions, and sharing options
- **Progress Indicators**: Visual progress bars for story sequences

## Technical Features

### Media Handling
- **Format Support**: Images (JPEG, PNG, HEIC) and videos (MP4, MOV)
- **Compression**: Automatic media optimization for performance
- **Duration Limits**: Configurable maximum duration for video stories
- **Quality Settings**: Adaptive quality based on network conditions

### Performance Optimization
- **Lazy Loading**: Stories load on-demand for better performance
- **Caching Strategy**: Intelligent caching of viewed story content
- **Preloading**: Smart preloading of upcoming stories
- **Memory Management**: Efficient cleanup of story resources

### Privacy and Security
- **Audience Control**: Granular control over story visibility
- **Content Moderation**: Built-in content filtering and reporting
- **Privacy Settings**: User-controlled privacy preferences
- **Data Protection**: Secure handling of story content and metadata

## Implementation Examples

### Basic Story Integration

<CodeGroup>
```swift iOS
import AmityUIKit

class StoriesViewController: UIViewController {
    override func viewDidLoad() {
        super.viewDidLoad()
        
        // Add story creation button
        let createButton = UIBarButtonItem(
            barButtonSystemItem: .camera,
            target: self,
            action: #selector(createStory)
        )
        navigationItem.rightBarButtonItem = createButton
        
        // Setup story feed
        setupStoryFeed()
    }
    
    @objc private func createStory() {
        let storyCreationVC = AmityCreateStoryViewController(
            targetId: currentCommunityId,
            targetType: .community
        )
        present(storyCreationVC, animated: true)
    }
}
```

```kotlin Android
class StoriesActivity : AppCompatActivity() {
    override fun onCreate(savedInstanceState: Bundle?) {
        super.onCreate(savedInstanceState)
        setContentView(R.layout.activity_stories)
        
        // Setup story creation
        findViewById<FloatingActionButton>(R.id.fab_create_story).setOnClickListener {
            val intent = AmityCreateStoryActivity.newIntent(
                context = this,
                targetId = currentCommunityId,
                targetType = AmityStoryTargetType.COMMUNITY
            )
            startActivity(intent)
        }
        
        // Setup story viewing
        setupStoryViewing()
    }
}
```

```typescript React Native
import React from 'react';
import { AmityStoriesComponent } from '@amityco/react-native-ui-kit';

export const StoriesScreen: React.FC = () => {
  const handleStoryCreate = () => {
    navigation.navigate('StoryCreation', {
      targetId: currentCommunityId,
      targetType: 'community'
    });
  };

  return (
    <AmityStoriesComponent
      onCreateStory={handleStoryCreate}
      onStoryPress={(story) => {
        navigation.navigate('StoryViewer', { storyId: story.storyId });
      }}
    />
  );
};
```
