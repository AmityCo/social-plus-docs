### Configuration-Based Customization

Customize your entire UIKit with a single `config.json` file:

```json
{
  "preferred_theme": "default",
  "theme": {
    "light": {
      "primary_color": "#1054DE",
      "secondary_color": "#292B32",
      "background_color": "#FFFFFF"
    },
    "dark": {
      "primary_color": "#1054DE",
      "secondary_color": "#292B32", 
      "background_color": "#191919"
    }
  },
  "customizations": {
    "select_target_page/*/*": {
      "title": "Custom Title",
      "page_theme": {
        "light_theme": {
          "primary_color": "#1D1234"
        }
      }
    }
  }
}
```

## Get Started in 5 Minutes

<Steps>
  <Step title="1. Choose Your Platform">
    Pick your development stack and install UIKit:

    <CodeGroup>

    ```bash iOS (CocoaPods)
    pod 'SocialPlusUIKit'
    ```

    
    ```bash Android (Gradle)
    implementation 'co.amity:social-plus-uikit:4.0.0'
    ```

    
    ```bash React Native
    npm install @amityco/social-plus-uikit-react-native
    ```

    
    ```bash Flutter
    flutter pub add amity_uikit
    ```

    
    ```bash Web (React)
    npm install @amityco/social-plus-uikit-web
    ```

    </CodeGroup>
  </Step>
  <Step title="2. Initialize UIKit">
    Configure authentication with your API credentials:

    <CodeGroup>

    ```swift iOS
    AmityUIKit.setup(apiKey: "YOUR_API_KEY")
    ```

    
    ```kotlin Android
    AmityUIKit.setup(apiKey = "YOUR_API_KEY")
    ```

    
    ```javascript React Native
    AmityUIKit.setup({ apiKey: 'YOUR_API_KEY' });
    ```

    </CodeGroup>
  </Step>
  <Step title="3. Add Your First Component">
    Start with a simple social feed:

    <CodeGroup>

    ```swift iOS
    let feedView = AmityFeedViewController()
    present(feedView, animated: true)
    ```

    
    ```kotlin Android
    val feedFragment = AmityFeedFragment()
    supportFragmentManager.beginTransaction()
        .replace(R.id.container, feedFragment)
        .commit()
    ```

    
    ```jsx React Native
    import { AmityFeedComponent } from '@amityco/social-plus-uikit-react-native';
    
    function App() {
      return <AmityFeedComponent />;
    }
    ```

    </CodeGroup>
  </Step>
  <Step title="4. Customize & Ship">
    Apply your brand colors and you're ready to go\!

    ```json config.json
    {
      "theme": {
        "light": {
          "primary_color": "#1054DE",
          "secondary_color": "#292B32"
        }
      }
    }
    ```
  </Step>
</Steps>
