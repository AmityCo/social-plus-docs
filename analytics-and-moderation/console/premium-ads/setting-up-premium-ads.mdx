## Creation Workflow

<Steps>
  <Step title="Start">Click Create Ads (ensure Advertiser Profile exists).</Step>
  <Step title="Select Profile">Choose advertiser identity (brand entity).</Step>
  <Step title="Upload Media">Add image (≤30MB) in proper ratio (9:16 Stories / 1:1 Post & Comment).</Step>
  <Step title="Add Copy">Headline, description, optional body text.</Step>
  <Step title="CTA (Optional)">Configure button label + destination URL.</Step>
  <Step title="Placement">Choose eligible placements (Post / Story / Comment).</Step>
  <Step title="Schedule">Set required start date & optional end date.</Step>
  <Step title="Review & Publish">Confirm preview → submit.</Step>
</Steps>

## Media & Copy Guidelines

<AccordionGroup>
  <Accordion title="Ratios">
    - 9:16: Fullscreen vertical, Stories / Reels.
    - 1:1: Square feed & comment surface.
  </Accordion>
  <Accordion title="File Constraints">Images ≤30MB; follow image handling spec for optimization & clarity.</Accordion>
  <Accordion title="Copy Structure">Headline (concise) → Description (value prop) → Optional body (context / qualifiers).</Accordion>
  <Accordion title="CTA Best Practices">Use action verbs, align label with destination (e.g., "Download", "Learn More").</Accordion>
  <Accordion title="Brand Consistency">Ensure tone matches brand profile settings; avoid mismatched capitalization styles.</Accordion>
</AccordionGroup>

## Editing Ads

<Steps>
  <Step title="Locate">In Ads list, click Edit on target ad.</Step>
  <Step title="Modify Fields">Adjust profile, name, ratio, copy, CTA, placement.</Step>
  <Step title="Preview">Verify updated creative & layout.</Step>
  <Step title="Save">Click Save Changes to apply.</Step>
</Steps>

## Deleting Ads

<Steps>
  <Step title="Open Edit">Click Edit for selected ad.</Step>
  <Step title="Scroll">Go to bottom of edit screen.</Step>
  <Step title="Delete">Click Delete Ad and confirm irreversible removal.</Step>
</Steps>

## Advertiser Profiles

<Info>Advertiser Profiles centralize legal identity & branding context used across multiple ads. Editing a profile updates all its associated ads.</Info>

<CardGroup cols={3}>
  <Card title="Create Profile" icon="plus">Establish advertiser entity</Card>
  <Card title="Edit Profile" icon="pencil">Update naming or compliance fields</Card>
  <Card title="Delete Profile" icon="trash">Remove unused identity (0 active ads)</Card>
</CardGroup>

### Creating a Profile

<Steps>
  <Step title="Navigate">Ads → Settings → Advertiser Profile tab.</Step>
  <Step title="Create">Click Create Profile.</Step>
  <Step title="Input Details">Enter Display Name & Official / Legal Name.</Step>
  <Step title="Save">Click Save Profile.</Step>
</Steps>

{/* Removed missing image: Profile Creation */}

### Editing a Profile

<Steps>
  <Step title="Open Settings">Ads → Settings → Advertiser Profile.</Step>
  <Step title="Select Profile">Click Edit Profile on target.</Step>
  <Step title="Change Fields">Adjust names / legal identifiers.</Step>
  <Step title="Save Updates">Apply changes; associated ads reflect updates.</Step>
</Steps>

<Warning>Editing a profile propagates across all linked ads—review large-scale naming shifts carefully.</Warning>

{/* Removed missing image: Edit Profile */}

### Deleting a Profile

<Steps>
  <Step title="Open Tab">Ads → Settings → Advertiser Profile.</Step>
  <Step title="Select">Click Edit Profile next to target.</Step>
  <Step title="Validate Usage">Ensure 0 active ads linked.</Step>
  <Step title="Delete">Click Delete Profile and confirm.</Step>
</Steps>

<Info>Profiles with any active or historical active ads in flight cannot be deleted until ads are retired.</Info>

## Best Practices

<AccordionGroup>
  <Accordion title="Creative Hygiene">Retire outdated seasonal creatives promptly; archive for audit if required.</Accordion>
  <Accordion title="Scheduling Discipline">Avoid overlapping campaigns with identical objectives targeting the same placement—prevents internal competition.</Accordion>
  <Accordion title="Profile Governance">Use standardized naming (Brand_Campaign_Q#_Year) for traceability.</Accordion>
  <Accordion title="CTA Integrity">Ensure destination URLs are live, secure (HTTPS), and aligned with label expectation.</Accordion>
  <Accordion title="Media Quality">Maintain resolution suitable for largest display; avoid text-heavy imagery.</Accordion>
</AccordionGroup>

## Troubleshooting

| Issue | Likely Cause | Resolution |
|-------|--------------|------------|
| Cannot create ad | Missing advertiser profile / permission | Create profile or adjust role |
| Image rejected | Exceeds size / unsupported ratio | Compress or resize to 9:16 / 1:1 |
| CTA not saving | Invalid URL format | Use full HTTPS URL | 
| Placement missing | Not supported for current ratio | Adjust ratio or choose valid placement |
| Profile delete blocked | Active ads exist | Retire or delete linked ads first |
| Schedule not applying | Missing start date | Set valid start (and optional end) |
