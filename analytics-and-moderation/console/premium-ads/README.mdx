## Display Configuration

<Tabs>
  <Tab title="Frequency Settings">
    ### Ad Display Modes

    <AccordionGroup>
      <Accordion title="Fixed Frequency Mode">
        **Predictable Ad Pattern:**
        - Ads appear at regular, predetermined intervals
        - Example: "Every 5 posts, 1 ad will appear"
        - Ensures consistent ad exposure
        - Predictable revenue generation
        - Easy performance forecasting

        **Configuration Options:**
        ```json
        {
          "displayMode": "fixed_frequency",
          "frequency": {
            "posts": 5,
            "ads": 1
          },
          "fallback": {
            "minimumContent": false,
            "showSingleAd": true
          }
        }
        ```

        **Behavior Rules:**
        - If fewer posts than set frequency, one ad still displays
        - No ads shown when no content exists
        - May result in duplicate ad viewing
        - Adjustable duplicate prevention settings available

        <Warning>
        Fixed frequency may lead to ad repetition. Configure duplicate prevention to maintain user experience quality.
        </Warning>
      </Accordion>

      <Accordion title="Time Window Mode">
        **Time-Based Ad Control:**
        - Ads appear once within specified time periods
        - Example: "1 ad every 10 minutes maximum"
        - Prevents ad fatigue and oversaturation
        - Maintains user engagement quality
        - Flexible timing configurations

        **Configuration Options:**
        ```json
        {
          "displayMode": "time_window",
          "timeWindow": {
            "duration": 600,
            "unit": "seconds",
            "maxAds": 1
          },
          "resetConditions": {
            "userSession": true,
            "contentRefresh": false
          }
        }
        ```

        **Advantages:**
        - Prevents ad oversaturation
        - Maintains user experience quality
        - Reduces ad blindness
        - Improves ad effectiveness
        - Better long-term user retention
      </Accordion>

      <Accordion title="Advanced Scheduling">
        **Smart Timing Features:**
        - Peak hours optimization
        - User activity-based scheduling
        - Geographic timezone adjustments
        - Seasonal campaign timing
        - A/B testing for optimal timing

        **Scheduling Options:**

      <Accordion title="Priority Scoring Algorithm">

        - Each ad receives priority points based on performance
        - First-time ads receive higher initial scores
        - Frequently shown ads receive lower priority over time
        - Balances ad exposure across campaigns
        - Prevents oversaturation of successful ads

        ```json

          "scoringFactors": {

            "budget": 0.15,
            "targeting": 0.1

          "scoreCalculation": {
            "base": 100,
        ```
