# User Social History

<Info>
The User Social History feature provides community managers, moderators, and admins with a comprehensive view of user activity across all communities. This feature helps ensure informed, transparent, and consistent moderation decisions while unlocking valuable insights into user behavior.
</Info>

## Overview

<CardGroup cols={2}>
  <Card title="User Analytics" icon="chart-line">
    Instantly view user statistics including posts, comments, reactions, and community participation alongside flagged content and moderation actions
  </Card>
  <Card title="Activity Timeline" icon="clock">
    Access a complete timeline of user posts, comments, and interactions including flagged and deleted content
  </Card>
  <Card title="Community Engagement" icon="users">
    View communities a user belongs to, their roles, and participation levels to assess influence and engagement
  </Card>
  <Card title="Moderation History" icon="shield">
    Track past moderation actions over time to detect behavioral patterns and ensure fair, data-driven decisions
  </Card>
</CardGroup>

## Key Features

<Tabs>
  <Tab title="User Assessment">

## Getting Started

<Steps>
  <Step title="Access User Management">
    Navigate to **"Manage Users"** in the Social+ Console
  </Step>
  <Step title="Select User">
    Click on any user row to open their social history profile
  </Step>
  <Step title="Review Information">
    Analyze user statistics, activity, and moderation history
  </Step>
  <Step title="Take Action">
    Make informed moderation decisions based on comprehensive data
  </Step>
</Steps>

### Alternative Access Methods

<AccordionGroup>
  <Accordion title="From Posts and Comments">
    Click on a user's name from the Posts and Comments management page to access their social history directly.
  </Accordion>
  <Accordion title="From Community Members">
    Access user social history by clicking on a user's name from the Member tab in any Community page.
  </Accordion>
</AccordionGroup>

## Best Practices

<AccordionGroup>
  <Accordion title="Effective User Assessment">
    **Pre-Action Review**
    - Always review user statistics before diving into detailed history
    - Check for patterns in flagged content and moderation history
    - Consider overall engagement metrics when making moderation decisions
    - Assess user's positive contributions alongside negative behavior

    **Contextual Analysis**
    - Review activity across multiple communities for complete picture
    - Consider timing and frequency of problematic behavior
    - Evaluate user's response to previous moderation actions
    - Assess potential for rehabilitation vs. continued issues
  </Accordion>
  <Accordion title="Consistent Moderation">
    **Fair Treatment Standards**
    - Use complete activity history to ensure fair treatment across all users
    - Review past moderation actions before implementing new ones
    - Consider context from multiple communities when evaluating behavior
    - Apply consistent standards regardless of user's community status

    **Documentation and Transparency**
    - Document reasoning for all moderation decisions
    - Maintain clear records of escalation processes
    - Ensure moderation actions align with community guidelines
    - Provide users with clear explanations for actions taken
  </Accordion>
  <Accordion title="Community Management">
    **Positive Engagement**
    - Identify valuable contributors across different engagement types
    - Recognize users who contribute positively to community discussions
    - Monitor user participation patterns across multiple communities
    - Use engagement data to support decisions about community roles

    **Proactive Management**
    - Use historical data to predict potential issues
    - Implement preventive measures for at-risk users
    - Provide guidance and support for users showing concerning patterns
    - Foster positive community culture through recognition and rewards
  </Accordion>
</AccordionGroup>
