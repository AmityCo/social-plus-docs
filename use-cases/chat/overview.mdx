## Channel Types at a Glance

Each chat use case maps to a different channel type in the SDK:

| Channel Type | Best For | Visibility | Max Members |
|---|---|---|---|
| **Live** | Livestream chat, event Q&A, broadcast commentary | Public | Unlimited |
| **Community** | Team rooms, Discord-style channels, community chat | Public or Private | Configurable |
| **Conversation** | 1:1 DMs, small group chats, support tickets | Private | ~20 (small group) |
